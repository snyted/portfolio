Todo List

Desenvolvi esta aplicação como parte do meu aprendizado e prática em desenvolvimento web. Meu objetivo era criar uma ferramenta útil que pudesse ser utilizada por qualquer pessoa para melhorar a gestão de suas tarefas diárias.

Funcionalidades Principais

  • Adicionar Tarefas: Interface simples para adicionar novas tarefas com descrição.
  • Marcar como Concluída: Opção para marcar tarefas como concluídas.
  • Remover Tarefas: Possibilidade de remover tarefas da lista.
  • Persistência de Dados: As tarefas são armazenadas localmente, mantendo-se após o fechamento do navegador.

Tecnologias Utilizadas

Gerador de Senha

A ideia para este projeto surgiu da necessidade crescente de criar senhas robustas que protejam contas e dados pessoais contra ataques de força bruta e outras ameaças cibernéticas. A ferramenta visa facilitar a criação de senhas seguras para qualquer pessoa.

Funcionalidades Principais

  • Tamanho das senhas: Geração de senhas aleatórias de qualquer comprimento.
  • Opções de caracteres: Opção para incluir letras maiúsculas, minúsculas, números e símbolos.
  • Copiar Senha: Opção para copiar a senha gerada para o clipboard.

Jogo do Detona Ralph

Este projeto foi desenvolvido como parte do bootcamp da DIO (Digital Innovation One) para praticar habilidades de programação, lógica de jogos e interatividade com o usuário. A ideia era criar um jogo envolvente que não só fosse divertido, mas também um desafio em termos de lógica e design de interface.

Funcionalidades Principais

  • Clique para Pontuar: Ralph aparece em uma das nove janelas aleatoriamente, e o jogador deve clicar nele para ganhar pontos.
  • Sistema de Vidas: Cada clique errado reduz uma vida do jogador. O jogo termina quando todas as vidas são perdidas.
  • Animações e Sons: Efeitos de animação e som para tornar o jogo mais dinâmico e envolvente.
  • Pontuação: Exibição da pontuação atual do jogador em tempo real.
